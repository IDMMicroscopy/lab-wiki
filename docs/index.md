# Welcome to the Lab Wiki

This is the central knowledge base for the facility — Standard Operating Procedures (SOPs), equipment guides, training materials, and policies.

## Quick links

- [SOPs](sops/index.md) — step-by-step procedures for lab tasks and instruments
- [Equipment](equipment/index.md) — instrument manuals, booking notes, maintenance logs
- [Training](training/index.md) — onboarding materials and competency checklists
- [Policies](policies/index.md) — safety, access, and administrative policies

## How to contribute

All content here is written in Markdown and stored in this repository. To add or edit a page:

1. Edit or create a `.md` file in the relevant `docs/` subfolder
2. Add it to the `nav:` section in `mkdocs.yml` if it's a new page
3. Commit and push — the site rebuilds automatically

See the `README.md` in the repository root for local setup and deployment instructions.

!!! tip "Not sure where something belongs?"
    Ask in the lab Slack/Teams channel, or just add it to the closest matching section — it's easy to move later.
